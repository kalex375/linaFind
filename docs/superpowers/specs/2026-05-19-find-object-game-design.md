# План реализации: Vue PWA "Найди предмет" для Android-планшета

## Summary

Сделать offline-first детскую игру "найди предмет" для Android-планшета. Игра работает как PWA, запускается с главного экрана, после первого открытия работает в самолете без интернета. Первая версия содержит 5 встроенных оригинальных уровней, основной режим экрана landscape, язык по умолчанию португальский, переключение на английский и украинский.

В v1 также нужен скрытый dev/admin редактор уровней. Через него добавляются начальные картинки: загружается сцена, выделяются предметы, задаются переводы, роли, подписи и аудио. Для предметов в списке поиска редактор автоматически вырезает мини-картинку из выделенной области сцены и использует ее в боковой панели.

AI-создание уровней не реализуется в v1, но модель данных должна быть совместима с будущим AI/родительским режимом.

## Key Implementation Changes

- Создать приложение на `Vue 3 + Vite + TypeScript`.
- Разрабатывать и проверять проект через Docker на Node `>=22.12`, чтобы не зависеть от локальных Node/NVM версий.
- Добавить `Pinia` для состояния игры: текущий уровень, язык, найденные предметы.
- Добавить PWA через `vite-plugin-pwa`:
  - precache app shell, встроенные уровни, изображения, thumbnails и аудио;
  - manifest для установки на Android home screen;
  - offline запуск после первого успешного открытия.
- Реализовать главный экран:
  - 5 крупных карточек уровней;
  - thumbnail уровня;
  - открытие выбранной сцены.
- Реализовать игровой экран:
  - landscape-first layout: большая сцена + боковая панель поиска;
  - portrait fallback: сцена сверху, список снизу;
  - переключатель `PT / EN / UK`;
  - без таймера, штрафов и сложных меню.
- Реализовать интерактивные предметы:
  - `search-target`: показывается в панели, участвует в прогрессе, отмечается найденным;
  - `learning-only`: кликается и озвучивается, но не входит в список поиска;
  - ошибочные клики не наказываются;
  - уровень завершается только после нахождения всех `search-target`.
- Реализовать аудио:
  - для встроенных уровней использовать локальные `mp3`/`ogg`;
  - если локального аудио нет, использовать Web Speech API как fallback;
  - если fallback недоступен, игра продолжает работать без озвучки.
- Реализовать скрытый редактор `/admin/levels`:
  - загрузка основной картинки сцены;
  - создание thumbnail уровня;
  - рисование прямоугольных зон предметов поверх сцены;
  - выбор роли предмета: `search-target` или `learning-only`;
  - ввод подписей `pt/en/uk`;
  - настройка `showLabelOnImage`;
  - загрузка локальных аудио по языкам;
  - авто-вырезка `thumbnailSrc` из `targetArea` для `search-target`;
  - локальный preview уровня;
  - экспорт уровня как JSON + assets для включения в встроенный контент.

## Data Model / Interfaces

```ts
type LanguageCode = 'pt' | 'en' | 'uk';

type LocalizedText = Record<LanguageCode, string>;

type Level = {
  id: string;
  title: LocalizedText;
  thumbnailSrc: string;
  imageSrc: string;
  source: 'built-in' | 'user' | 'ai';
  items: SceneItem[];
};

type SceneItem = {
  id: string;
  role: 'search-target' | 'learning-only';
  label: LocalizedText;
  showLabelOnImage: boolean;
  targetArea: {
    x: number;
    y: number;
    width: number;
    height: number;
  };
  thumbnailSrc?: string;
  audioSrc?: Partial<Record<LanguageCode, string>>;
};
```

## Assumptions

- Основная платформа: Android-планшет, Chrome/PWA.
- Основная ориентация: landscape.
- Первая версия содержит 5 встроенных уровней.
- Иллюстрации оригинальные, без защищенных персонажей и сканов книг.
- Язык по умолчанию: португальский.
- Дополнительные предметы кликаются и озвучиваются только если явно добавлены в редакторе.
- В v1 нет backend, публичного родительского режима и AI-генерации.
- Скрытый редактор нужен как инструмент подготовки встроенных уровней, а не как часть детского игрового сценария.
